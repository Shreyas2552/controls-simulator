PID Control Law

The PID controller computes:
u(t) = Kp·e + Ki·∫e dt + Kd·(de/dt)

P — reacts to current error (fast, leaves offset)
I — eliminates steady-state error (accumulates history)
D — anticipates future error via rate of change (damps overshoot)

Derivative filter N: practical D-term uses Kd·N·s/(s+N). High N ≈ ideal derivative but noisy; low N = smooth but less effective. Typical: 10–200.

LQR — Linear Quadratic Regulator

Minimises infinite-horizon cost:
J = ∫₀^∞ (x'Qx + u'Ru) dt

Solve Algebraic Riccati Equation:
A'P + PA − PBR⁻¹B'P + Q = 0

Optimal gain: K = R⁻¹B'P, control law: u = −Kx

Kalman Filter (dual of LQR)

Solves: APe + PeA' − PeC'Rn⁻¹CPe + Qn = 0
Observer gain: L = PeC'Rn⁻¹
State estimate: x̂̇ = Ax̂ + Bu + L(y − Cx̂)

Separation Principle (LQG)

LQR + Kalman combined. Design independently:
CL eigs = eig(A−BK) ∪ eig(A−LC)

Q: larger → tighter state tracking (more control effort)
R: larger → cheaper control (slower response)
Qn/Rn: process/measurement noise — tunes observer aggressiveness

MPC — Finite Horizon Optimal Control

At each step k, solve:
min J = Σᵢ Qy·(y_{k+i|k}−r)² + Σⱼ Ru·u_{k+j}²
subject to u_min ≤ u ≤ u_max

Apply u₀ (receding horizon), repeat at k+1.

Prediction Matrices (unconstrained solution)

Y = Φ·x + Ψ·U
U_opt = (Qy·ΨᵀΨ + Ru·I)⁻¹ · Qy·Ψᵀ·(r·1 − Φ·x)

Φ[i] = C·A_d^{i+1} · x (free response)
Ψ[i,j] = C·A_d^{i−j}·B_d (forced response)

Smith Predictor

Removes delay from the control loop. Error signal:
e_SP = r − y − (G_m·e^{−Ls} − G_m)·u
Effective CL: T(s) = C(s)G(s)/(1+C(s)G(s)) · e^{−Ls}
Delay only on output — NOT in characteristic equation. Much easier to tune.

FOPDT System Identification

G(s) = K·e^{−Ls}/(τs+1) — three parameters from step response:
K = y_ss/u_step | τ from 63.2% crossing | L from 5% crossing
Cohen-Coon auto-tuning: Kp = (1.35/K)·(τ/L)·(1+0.185r), r = L/τ

Cascade, IMC/SIMC, Feedforward & SMC Simulations

Interactive simulations coming in v2.1.

Cascade Control

Inner loop bandwidth ≥ 3× outer loop. Rejects disturbances entering between loops before they reach the outer output.

SIMC Tuning (Skogestad IMC)

Single tuning parameter λ (desired CL time constant):
Kp = τ/(K·(λ+θ)), Ti = min(τ, 4(λ+θ))
λ = θ → aggressive, λ = 5τ → conservative. Best single-parameter PID method.

Sliding Mode Control

Sliding surface s = ė + c·e. Control drives state to surface (s=0) then slides to origin.
u = u_eq + K·sat(s/φ). Boundary layer φ eliminates chattering at expense of steady-state error.
